# @zentus/eslint-config-zen

## Installation
```bash
npm install @zentus/eslint-config-zen eslint@7
```

## Setup
Create a `.eslintrc.cjs` in the root directory:

```javascript
require('@zentus/eslint-config-zen/patch')

module.exports = {
  extends: '@zentus/eslint-config-zen'
}
```
